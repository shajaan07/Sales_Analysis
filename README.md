Developed a Sales Dashboard using Excel to track sales trends, top-performing products, and regional performance. Used data cleaning, pivot tables, and visualizations to generate business insights.

# 📊 Sales Dashboard Project

## 📌 Overview

This project is a Sales Dashboard created using Excel.
It helps to analyze sales performance, identify trends, and generate useful business insights.

---

## 🎯 Objective

* To analyze sales data and understand business performance
* To identify top-performing products and regions
* To track key metrics like revenue, profit, and sales growth

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Data Cleaning Techniques

---

## 📂 Dataset

The dataset contains sales-related information such as:

* Order Date
* Product Name
* Region
* Sales Amount
* Quantity
* Profit

---

## 📊 Key Features

* Interactive dashboard with filters
* Sales trend analysis over time
* Region-wise performance analysis
* Top-selling products identification
* KPI metrics (Total Sales, Profit, Quantity)

---

## 🔍 Process

1. Cleaned the raw data (removed null values, formatted data)
2. Created pivot tables for analysis
3. Designed charts for visualization
4. Built an interactive dashboard using slicers

---

## 📈 Insights

* Identified top-performing regions and products
* Found sales trends across different time periods
* Helped in understanding business growth patterns

---

## 🚀 Conclusion

This project shows how Excel can be used to transform raw data into meaningful insights and support decision-making.
